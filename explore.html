About Autonomys DSN

The Distributed Storage Network (DSN) permanently stores data across thousands of farmers worldwide. Each file is:

  • Content-addressed - identified by a unique CID (Content Identifier)
  • Replicated - stored across multiple nodes for redundancy
  • Verified - cryptographic proofs ensure data integrity
  • Permanent - stored as long as the network exists
